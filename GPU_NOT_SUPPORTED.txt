================================================================================
                  GPU FAN CONTROL NOT SUPPORTED
================================================================================

If you're getting "Failed to set fan speed" error, your GPU likely doesn't
support software-based fan control through LibreHardwareMonitor.

================================================================================
WHY THIS HAPPENS
================================================================================

1. GPU MODEL LIMITATION
   ‚Ä¢ Some GPU models have hardware-locked fan curves
   ‚Ä¢ Manufacturer disabled software control in firmware
   ‚Ä¢ Budget/OEM cards often lack this feature

2. LAPTOP GPUs
   ‚Ä¢ Most laptop GPUs have locked fan control
   ‚Ä¢ Controlled by laptop manufacturer's EC (Embedded Controller)
   ‚Ä¢ Fan control tied to system cooling, not just GPU

3. MANUFACTURER RESTRICTIONS
   ‚Ä¢ Some brands lock fan control to their own software
   ‚Ä¢ Requires specific tools (MSI Afterburner, EVGA Precision, etc.)

4. BIOS/VBIOS SETTINGS
   ‚Ä¢ Fan control may be disabled in GPU BIOS
   ‚Ä¢ Some manufacturers ship with this disabled

   ‚Ä¢ Better case airflow helps GPU cooling
   ‚Ä¢ More case fans = cooler GPU

   ‚Ä¢ Aftermarket GPU coolers available
   ‚Ä¢ Arctic Accelero, Raijintek, etc.
   ‚Ä¢ Requires technical skills

   ‚Ä¢ Some motherboards allow GPU fan control

   ‚Ä¢ Use MSI Afterburner or manufacturer software
