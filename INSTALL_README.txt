================================================================================
  GPU FAN CONTROLLER v2.0 - INSTALLATION INSTRUCTIONS
================================================================================

WHAT YOU HAVE:
--------------
This package contains everything needed to create a professional Windows 
installer for GPU Fan Controller.

QUICK START - CREATE INSTALLER:
--------------------------------
1. Install Inno Setup (free):
   Download: https://jrsoftware.org/isdl.php
   
2. Install .NET 6.0 SDK (free):
   Download: https://dotnet.microsoft.com/download/dotnet/6.0

3. Run: build-installer.bat

4. Find installer: installer_output\GPUFanController_Setup_v2.0.exe

5. Distribute the installer file to users!

WHAT THE INSTALLER DOES:
-------------------------
✓ Installs GUI and Console versions
✓ Creates Start Menu folder with shortcuts
✓ Optional desktop shortcuts
✓ Includes all documentation
✓ Adds uninstaller to Control Panel
✓ Requests administrator privileges automatically
✓ Self-contained (no .NET needed on user PCs)

FOR END USERS:
--------------
If you received the installer file (GPUFanController_Setup_v2.0.exe):

1. Double-click the installer
2. Click "Yes" on UAC prompt (admin access required)
3. Follow the installation wizard
4. Choose optional desktop shortcuts
5. Click Install
6. Launch from Start Menu or desktop

REQUIREMENTS:
-------------
- Windows 10 or Windows 11 (64-bit)
- Administrator privileges
- NVIDIA, AMD, or Intel GPU with fan control support

SAFETY WARNING:
---------------
⚠ This software controls GPU hardware fan speeds
⚠ Improper use can cause overheating and hardware damage
⚠ Always monitor GPU temperatures
⚠ Never set fan speed below 30% without monitoring
⚠ Use at your own risk

TO UNINSTALL:
-------------
Method 1: Settings > Apps > GPU Fan Controller > Uninstall
Method 2: Start Menu > GPU Fan Controller > Uninstall
Method 3: Control Panel > Programs and Features
