================================================================================
üîß TROUBLESHOOTING BUILD ISSUES
================================================================================

Error: "dotnet is not recognized"
Fix: Install .NET 6.0 SDK, restart computer, try again

Error: "Inno Setup not found"
Fix: Install Inno Setup to default location, try again

Error: Build fails with compilation errors
Fix: 
  1. Open Command Prompt as Administrator
  2. Navigate to project folder
  3. Run: dotnet clean
  4. Run: dotnet restore
  5. Run: build-everything.bat

Error: GPU not detected when testing
Fix: Right-click .exe ‚Üí "Run as administrator"

Error: Installer won't run on another PC
Fix:
  - Ensure target PC is Windows 10/11 64-bit
  - Disable antivirus temporarily
  - Run as administrator
