================================================================================
TROUBLESHOOTING
================================================================================

Problem: "dotnet is not recognized"
Solution: 
  - Install .NET 6.0 SDK (see Step 1A)
  - Restart computer
  - Try again

Problem: "Inno Setup not found"
Solution:
  - Install Inno Setup (see Step 1B)
  - Make sure it's installed to default location:
    C:\Program Files (x86)\Inno Setup 6\
  - Try again

Problem: Build fails with errors
Solution:
  - Open command prompt as administrator
  - Navigate to project folder
  - Run: dotnet clean
  - Run: dotnet restore
  - Run: build-everything.bat

Problem: GPU not detected when running
Solution:
  - MUST run as administrator
  - Right-click the .exe file
  - Select "Run as administrator"
  - Update GPU drivers

Problem: Installer won't run on another PC
Solution:
  - Target PC must be Windows 10/11 64-bit
  - Target PC must have administrator privileges
  - If antivirus blocks, add exception
  - Standalone .exe doesn't need .NET installed

================================================================================
MANUAL BUILD (Advanced)
================================================================================

If you prefer command line:

1. Restore packages:
   dotnet restore

2. Build GUI:
   dotnet build GPUFanController.csproj -c Release

3. Build Console:
   dotnet build GPUFanControllerConsole.csproj -c Release

4. Publish standalone GUI:
   dotnet publish GPUFanController.csproj -c Release -r win-x64 ^
     --self-contained true -p:PublishSingleFile=true ^
     -p:IncludeNativeLibrariesForSelfExtract=true

5. Publish standalone Console:
   dotnet publish GPUFanControllerConsole.csproj -c Release -r win-x64 ^
     --self-contained true -p:PublishSingleFile=true ^
     -p:IncludeNativeLibrariesForSelfExtract=true

6. Create installer:
   "C:\Program Files (x86)\Inno Setup 6\ISCC.exe" installer.iss
