                  HOW TO ADD A COOL ICON TO YOUR INSTALLER
================================================================================

OPTION 1: Use a Free Online Icon Converter (EASIEST - 2 minutes)
================================================================

Step 1: Find or Create an Image
--------------------------------
You need a square image (PNG or JPG) related to:
- GPU cooling
- Computer fan
- Temperature control
- Tech/hardware theme

Free icon sources:
• Flaticon.com - Search "gpu fan" or "cooling"
• Icons8.com - Search "fan" or "gpu"
• Noun Project - Search "cooling fan"
• Or create your own image (256x256px recommended)

Step 2: Convert to .ICO Format
-------------------------------
Visit: https://icoconverter.com/

1. Click "Choose File" and select your image
2. Select icon sizes: ✓ 16x16, ✓ 32x32, ✓ 48x48, ✓ 256x256
3. Click "Convert ICO"
4. Download the .ico file

Step 3: Replace the Icon File
------------------------------
1. Rename your downloaded file to: app.ico
2. Copy it to the project folder (replace existing placeholder)
3. Run: build-everything.bat again

Done! Your installer will now have a custom icon!


OPTION 2: Use GIMP (Free Software)
===================================

Step 1: Install GIMP
--------------------
Download from: https://www.gimp.org/downloads/

Step 2: Create Icon
-------------------
1. Open GIMP
2. File > New
3. Set size: 256x256 pixels
4. Design your icon (use fan/cooling theme)
5. File > Export As
6. Save as: app.ico
7. Choose ICO format
8. Select multiple sizes when prompted

Step 3: Copy to Project
-----------------------
Copy app.ico to the project folder
Run: build-everything.bat

RECOMMENDED ICON DESIGN ELEMENTS
=================================

Colors:
• Primary: Cyan/Blue (#00BFFF, #1E90FF) - cooling theme
• Secondary: White/Light Gray - for contrast
• Accent: Dark Blue/Black - for depth

Style:
• Flat design or slight 3D effect
• Clean, modern look
• Simple enough to recognize at 16x16px
• Professional, not cartoonish

Elements:
• Cooling fan with 3-4 blades
• Optional: Circular background
• Optional: Small temperature icon or arrow
• Keep it simple!


QUICK TEMPORARY ICON (Use Existing Icons)
==========================================

If you want to build NOW without creating custom icon:

Windows has built-in icons you can use temporarily:
1. Find a .ico file in: C:\Windows\System32\
2. Copy any .ico file (like imageres.dll icons)
3. Use an icon extractor tool
4. Name it app.ico

Or skip the icon entirely:
1. The installer will use default Windows icon
2. Still looks professional
3. Add custom icon later


AFTER ADDING YOUR ICON
=======================

1. Make sure the file is named: app.ico
2. Place it in the project root folder
3. Edit installer.iss, uncomment line 22:
   
   From:  ; SetupIconFile=app.ico
   To:    SetupIconFile=app.ico

4. Run: build-everything.bat

Your installer will now show your custom icon!


ICON FILE REQUIREMENTS
=======================

Format: .ICO (Windows Icon)
Required sizes:
• 16x16 pixels - Taskbar, small icons
• 32x32 pixels - Standard icons
• 48x48 pixels - Large icons
• 256x256 pixels - High DPI displays

File size: Usually 20-100 KB
Transparency: Supported (recommended for modern look)

TESTING YOUR ICON
==================

After adding icon and rebuilding:

1. Check installer file:
   Right-click: installer_output\GPUFanController_Setup_v2.0.exe
   Your icon should appear

2. Install and check:
   After installation, check:
   - Desktop shortcut (if created)
   - Start Menu shortcut
   - Taskbar when running
   - Control Panel > Programs

All should show your custom icon!
