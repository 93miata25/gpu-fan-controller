TROUBLESHOOTING
================================================================================

If TEST_BUILD.bat says ".NET SDK not found":
- Close this window
- Open a NEW command prompt
- Try again
- If still not working, restart your computer

If build-everything.bat fails:
- Make sure you ran TEST_BUILD.bat first
- Check error messages
- Try running individual build scripts (build.bat)

If "dotnet not recognized" error:
- .NET SDK may not be in PATH yet
- Restart computer to refresh environment variables
- Or manually add to PATH
